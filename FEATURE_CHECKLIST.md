# 墨岩缠论分析系统 - 功能检查清单

## 📋 每次更新后必须检查的功能清单

### 🎯 **基础功能检查**

#### ✅ **1. 数据获取与处理**
- [ ] 股票代码输入正常工作
- [ ] 时间区间选择生效
- [ ] K线级别切换正常（1d, 1h, 30m, 15m等）
- [ ] 数据源切换和容错机制正常
- [ ] 数据完整性校验通过
- [ ] 无空数据或异常数据

#### ✅ **2. 图表基础显示**
- [ ] K线图正常显示（红涨绿跌）

- [ ] 颜色配色方案正确
